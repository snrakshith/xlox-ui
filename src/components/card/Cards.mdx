    # Card
    A component which is designed to present data in a container

    ## `How to use it ?`
    1. Import `Card` component from uikit.

    ```jsx
    import { Card } from "@ikuretechsoft/uikit/lib/components"
    ```

    ## Example
    - A working example, on how to use

    ```jsx
    import { Card } from "@ikuretechsoft/uikit/lib/components";

    const card = () => {
        return (
            <>
                    <Card
                        card={{}} />
                        header={<Card.Header />}
                        body={<Card.Body />}
                        footer={<Card.Footer />}
            </>
        );
    };

    export default card;

    ```
    ## `Component API` for card

    - The `header`

    ```jsx
    <Card.Header/>
    ```

    - The `body`
    ```jsx
    <Card.Body/>
    ```

    - The `footer`
    ```jsx
    <Card.Footer/>
    ```
